执行计划(Explain plan) 查询在 Oracle 中被执行的一个执行路径。

SQL 命令可以按功能分为：

  • 数据检索 select
  • 数据操作(DML，data manipulation language) insert、update、delete
  • 数据定义(DDL，data definition language) create、alter、drop、rename、truncate
  • 数据控制(DCL，data control language) grant、revoke
  • 事务处理 commit、rollback、savepoint、set transaction
